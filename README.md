# Weibo_Comment_Viewer

## Configuration

Create a local `.env` file in the project root before using the AI analysis or mail features.

You can start from `.env.example`.

Required variables:

- `DEEPSEEK_API_KEY`
- `SMTP_USER`
- `SMTP_PASS`

Optional variables:

- `DEEPSEEK_BASE_URL`
- `SMTP_SERVICE`
- `MAIL_FROM`
